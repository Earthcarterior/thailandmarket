Step 1 — เลือกวิธี Deploy

ทุกวิธีใช้ static hosting ไม่ต้องการ server — เลือกตามความสะดวก

✅ สถานะปัจจุบัน Audit ผ่าน 54/54 หน้า · ไม่มี broken links · ไม่มี ZenMart · ทุก Flow เชื่อมครบ · Mobile responsive ครบ

Netlify Drop
ฟรี
⏱ 2 นาที · ไม่ต้องสมัคร · CDN ทั่วโลก
  1. ไปที่ app.netlify.com/drop
  2. Zip ทุกไฟล์ในโฟลเดอร์ → drag & drop ลงหน้า
  3. ได้ URL ทันที เช่น tm-abc123.netlify.app
  4. (Optional) Settings → Domain → ใส่ domain ตัวเอง
เปิด Netlify Drop →
🐙
GitHub Pages
ฟรี
⏱ 5 นาที · ต้องมี GitHub account
  1. สร้าง repo ใหม่บน github.com
  2. Upload ทุกไฟล์ (drag & drop ได้ในเว็บ)
  3. Settings → Pages → Source: main branch
  4. URL: username.github.io/repo-name
สร้าง Repo →
▲
Vercel
ฟรี / $20
⏱ 3 นาที · เร็วมาก · เหมาะ Production frontend
  1. ติดตั้ง: npm i -g vercel
  2. รัน: vercel ในโฟลเดอร์
  3. Framework: Other · Root: ./
  4. URL: thailand-market.vercel.app
เปิด Vercel →
🔶
Cloudflare Pages
ฟรี
⏱ 5 นาที · CDN ไทย · DDoS protection
  1. ไปที่ pages.cloudflare.com
  2. Create project → Upload assets
  3. Upload ทุกไฟล์ (zip หรือ folder)
  4. URL: thailand-market.pages.dev
เปิด CF Pages →
Terminal — Git Deploy (Netlify / Vercel CLI)
# 1. เตรียม Git repository
git init && git add . && git commit -m "Thailand Market v1.0"

# 2a. Deploy ด้วย Netlify CLI
npm install -g netlify-cli
netlify deploy --prod --dir .

# 2b. หรือ deploy ด้วย Vercel CLI
npm install -g vercel
vercel --prod
ตั้งค่า Custom Domain (thailandmarket.co.th)
# DNS Settings ที่ Registrar (เช่น godaddy, namecheap, thnic)
Type: CNAME
Name: www
Value: (URL จาก Netlify/Vercel/CF Pages)
TTL: Auto (หรือ 3600)

# สำหรับ root domain (@)
Type: A
Name: @
Value: 75.2.60.5  (Netlify) หรือ 76.76.21.21 (Vercel)

Step 3 — ขั้นตอนถัดไป

Roadmap สู่ Production

จาก Static Prototype → Full Production Platform — แบ่งเป็น 3 Phase

ℹ️ ตอนนี้คืออะไร? ไฟล์ HTML เหล่านี้คือ UI Prototype — ทุก click ทำงานได้ ทุก flow เชื่อมกัน แต่ข้อมูลเป็น mock ยังไม่มี real backend, database, หรือ payment จริง
Phase 1
MVP Backend
เดือน 1–3 · ~฿150K–300K
  • NestJS + PostgreSQL API
  • JWT Auth + LINE Login
  • Omise Payment Gateway
  • AWS S3 รูปภาพสินค้า
  • Basic Seller Portal
  • Kerry API Integration
  • Firebase Push Notification
Hosting: ~$55/เดือน
Vercel + RDS + EC2 t3.medium
Phase 2
Growth Features
เดือน 3–6 · ~฿300K–500K
  • Elasticsearch Product Search
  • Flash Sale Engine
  • Live Shopping (WebRTC)
  • Loyalty Points System
  • Analytics Dashboard
  • Mobile App (React Native)
  • Seller Ads Platform
Hosting: ~$200/เดือน
+ Redis cluster + Elasticsearch
Phase 3
Scale & Optimize
เดือน 6–12 · ~฿500K+
  • Kubernetes + Auto-scaling
  • AI Recommendation Engine
  • B2B Procurement Portal
  • Multi-vendor Fulfillment
  • Fraud Detection
  • International Expansion
  • IPO Ready Reporting
Hosting: ~$1,000+/เดือน
AWS Multi-AZ + CDN + WAF
