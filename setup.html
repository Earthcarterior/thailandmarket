ตั้งค่าระบบ

กรอก Supabase URL และ API Key — ทำครั้งเดียว บันทึกในเบราว์เซอร์

1
ไปที่ supabase.com → สร้าง Project ใหม่ฟรี → เลือก Region: Southeast Asia
2
ไปที่ Project Settings → API → คัดลอก URL และ anon public key
3
ไปที่ SQL Editor → วางและรัน SQL จากไฟล์ supabase-setup.sql
4
ไปที่ Authentication → Users → Add user → สร้าง Admin account แรก
✅ บันทึกแล้ว กำลังพาไปหน้า Admin...
❌ กรุณากรอกข้อมูลให้ครบถ้วน
